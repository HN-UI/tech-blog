# 테크 블로그 프로젝트 기획서

> 레트로 게임 UI 느낌의 개인 테크 블로그
> 목적: 개발 실력 향상(연습) + 포트폴리오

---

## 1. 컨셉 / 디자인 방향

- **전체 무드**: 포켓몬 같은 레트로 게임 UI 느낌
- **폰트 전략**
  - 제목 / UI 요소: 픽셀 폰트 (예: `Press Start 2P`, 또는 무료 도트 한글 폰트 `둥근모`/`갈무리`)
  - 본문: 가독성 좋은 한글 폰트 (예: `Pretendard`, `Noto Sans KR`)
  - 이유: 픽셀 폰트를 본문 전체에 쓰면 눈이 피로하고 가독성 떨어짐

### 팔레트

| 용도 | 색상 |
|---|---|
| 메인 대화창 테두리 (진한 파랑) | `#3868C8` |
| 대화창 이너 테두리 (연한 파랑) | `#A8C4EE` |
| 소개 카드 테두리 (핑크/레드) | `#D9556E` |
| 소개 카드 이너 테두리 | `#F3B3C0` |
| 텍스트 | `#222222` |
| 검은 독 바 | `rgba(0, 0, 0, 0.72)` |

## 2. 페이지 구조 (라우트)

| 페이지 | 경로 | 내용 |
|---|---|---|
| 홈 | `/` | 도트 메인 화면 + 하단 커버플로우 독 |
| 글 상세 | `/posts/[slug]` | 실제 글 본문 |
| 카테고리 목록 | `/category/[name]` | 연구 / 개발 회고 / 일정별 모아보기 |
| 소개 | `/about` | 프로필 카드 |

### 카테고리
- 프로필
- 연구 회고
- 프로젝트
- 리포트
- 폴더

---

## 3. 메인 화면 (홈) 설계

### 3-1. 상단
- 도트(픽셀) 이미지 배경 → **나중에 채울 예정**

### 3-2. 하단 커버플로우 독 (핵심 인터랙션)
- 화면 좌우 끝까지 꽉 찬 반투명 검은 바
- 아이콘 항목: **프로필 / 프로젝트 / 연구 / 리포트 / 폴더**
  - 지금은 이모지, 나중에 도트 이미지로 교체
- 동작 방식:
  - 마우스/터치로 잡고 좌우 드래그 → 촤라라락 회전
  - 가운데 아이템: 가장 선명(opacity 1) + 가장 큼(scale 1)
  - 양옆(2, 4번째): 살짝 투명(0.55) + 조금 작게(0.82)
  - 맨 끝(1, 5번째): 더 투명(0.25) + 더 작게(0.68)
  - 놓으면 관성으로 미끄러지다가 가장 가까운 아이템이 가운데로 snap
  - 무한 루프 (끝까지 돌리면 다시 처음으로)

> ⚠️ 이 커버플로우는 순수 JS로 직접 구현하면 복잡함.
> Next.js 단계에서 `Embla Carousel` 라이브러리로 구현하는 게 편함.
> 지금 연습 단계에서는 정적 배치 + hover 효과 + CSS 애니메이션까지만 먼저.

## 4. 소개(About) 페이지 설계

프로필 카드 형식 (게임 트레이너 카드에서 영감받은 오리지널 디자인)

- **상단 타이틀 바**: 핑크 배경 + 별 아이콘 + "PROFILE CARD" 제목
- **본문**: 좌측 아바타 박스 + 우측 정보 리스트 (점선 구분선)
- **하단 배너**: 목표 한 줄

### 정보 항목
- 이름
- 시작한 날
- 작성한 글 수
- 관심 분야 / 기술 스택
- 목표 (한 줄)

## 5. 기술 스택

- **Next.js** (App Router) — 프론트 + 백엔드를 하나로
- **MDX** — 마크다운으로 글 작성 (`content/` 폴더에 `.mdx` 파일로 관리)
- **CSS** (또는 Tailwind CSS)
- 코드 하이라이팅: `rehype-pretty-code` 또는 `shiki`
- 배포: **Vercel**
- 버전 관리: **Git + GitHub**

---

## 6. 학습 로드맵

1. **HTML & CSS** — 웹 페이지 구조와 스타일 (박스 모델, Flexbox)
2. **JavaScript** — 동작과 상호작용 로직 (DOM 조작)
3. **React** — 컴포넌트, props, state
4. **Next.js** — 라우팅, 서버 기능
5. **Git & 배포** — 버전 관리, Vercel

> 각 단계 70%만 익히고 넘어가서, 블로그 만들며 부족한 부분 채우기
